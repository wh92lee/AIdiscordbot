====================================
  오딘 보스 알림봇 명령어 정리
====================================

[ 보스 처치 기록 ]
  !킬 [보스이름]
    → 현재 시각 기준 처치 기록, 리젠 시간 자동 계산 후 알림 등록
    예) !킬 니드호그

  !킬 [보스이름] [HH:MM]
    → 특정 시각에 처치한 것으로 기록 (늦게 입력할 때 사용)
    예) !킬 니드호그 14:30

────────────────────────────────────

[ 수동 리젠 예약 (점검 후 사용) ]
  !젠 [보스이름] [HH:MM]
    → 리젠 예상 시각을 직접 입력, 해당 시각에 알림
    → 점검 종료 후 보스가 랜덤 시간에 젠될 때 사용
    → 입력한 시각이 현재보다 이전이면 다음날로 자동 처리
    예) !젠 니드호그 15:30
    예) !젠 펜리르 22:00

  ※ !킬 vs !젠 차이
    !킬  → 처치 시각 + 리젠 시간 자동 계산  (일반 사냥 후 사용)
    !젠  → 리젠 예상 시각 직접 입력          (점검 후 수동 예약)

────────────────────────────────────

[ 알림 설정 ]
  !알림설정
    → 보스별 알림 ON/OFF 버튼 목록 출력
    → 버튼 클릭으로 즉시 토글 (🟢 ON / 🔴 OFF)
    → 설정은 boss_config.json 에 자동 저장 (봇 재시작 후에도 유지)
    → 알림 OFF 보스는 !킬 / !젠 입력 시 등록 거부

────────────────────────────────────

[ 현황 조회 ]
  !현황
    → 현재 리젠 대기 중인 보스 목록 출력
    → 남은 시간 짧은 순으로 정렬
    → 10분 이내 보스는 상단에 한 줄로 묶어서 표시
       예) 🚨 10분 이내 리젠: 드레이크 → 트롤

  !보스목록
    → bosses.txt 에 등록된 전체 보스 목록 출력
    → 각 보스의 리젠 시간과 알림 상태(🟢/🔴) 함께 표시

────────────────────────────────────

[ 알림 기능 ]
  리젠 알림 (자동)
    → 등록된 리젠 시각이 되면 @here 멘션과 함께 알림 발송
    → 알림 메시지에 [⚔️ 컷!] 버튼 포함
    → 컷 버튼 클릭 시 현재 시각 기준으로 다음 리젠 자동 등록
    → 버튼은 한 번만 클릭 가능, 누가 컷했는지 이름 표시

  5분 전 묶음 알림 (자동)
    → 리젠 시각이 10분 이내로 겹치는 보스가 2마리 이상일 경우
       가장 빠른 보스 기준 5분 전에 묶음 알림 발송
       예) ⚠️ 보스 리젠 5분 전!  니드호그 → 펜리르

────────────────────────────────────

[ 음성 알림 설정 ]
  !음성채널
    → 현재 입장 중인 음성 채널을 알림 채널로 설정
    → 이미 설정된 채널이 있으면 현재 설정 채널 이름 표시
    → 설정 후 보스 리젠 알림 / 5분 전 묶음 알림 시 TTS 음성으로 함께 알림

  !음성채널해제
    → 음성 알림 설정 해제 (텍스트 알림만 유지)

  ※ 음성 알림 사용 전 준비사항
    - 서버에 ffmpeg 설치 필요
    - pip install discord.py[voice] gTTS

────────────────────────────────────

[ 알림 취소 ]
  !취소 [보스이름]
    → 해당 보스의 대기 중인 리젠 알림 취소
    예) !취소 니드호그

────────────────────────────────────

[ 기타 ]
  !핑
    → 봇 응답 속도 확인

====================================
  파일 설명
  bosses.txt      → 보스 이름 / 리젠 시간 추가·수정
  boss_config.json → 알림 ON/OFF 상태 저장 (자동 생성)
